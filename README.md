# Positiv Consult 🇦🇹

Website für Positiv Consult - Digitalisierung, Automatisierung und KI-Beratung in Österreich.

**Live:** [positiv-consult.at](https://positiv-consult.at)

## 🚀 Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** CSS
- **Hosting:** GitHub Pages
- **CI/CD:** GitHub Actions

## 🛠️ Entwicklung

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build

# Production preview
npm run start
```

## 📦 Deploy

Automatisches Deploy bei Push zu `main`:
- Production: [positiv-consult.at](https://positiv-consult.at)
- PR Previews: Automatische Preview-URLs werden erstellt und beim Schließen gelöscht
